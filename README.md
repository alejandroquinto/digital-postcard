# Digital Postcard ✉️

A small web app that lets you write a postcard with handwriting fonts, a live map (Mapbox), and realistic paper textures — then export it as PNG or JPG.

## Preview
Open `index.html` in your browser, enter your note, choose a map location, and download your custom postcard.

Built with:
- HTML5 + CSS + JS (Canvas)
- Mapbox API
- PP Neue Machina & Magonda fonts
